# Skill Plans

This directory contains EVE Online skill plans that can be downloaded and used by CanIFly.

## Format

Each skill plan is a text file with the `.txt` extension. The format is:
```
Skill Name Level
```

Example:
```
CPU Management 5
Power Grid Management 5
```

## Adding New Plans

To add a new skill plan:
1. Create a new `.txt` file in this directory
2. Add skills in the format shown above
3. Commit and push to the repository

The plans will be automatically downloaded by CanIFly when the application starts.

## Common Skill Plans

- **Magic_14.txt** - The 14 core skills every pilot should train
- **Interceptor.txt** - Skills for flying interceptors effectively
- **Stealth_Bomber.txt** - Skills for covert ops and bombing runs
